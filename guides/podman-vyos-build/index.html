Building VyOS using Podman

April 26, 2023
linux networking

The VyOS developers only provide free access to nightly builds of VyOS, and not stable releases. If you want to run a stable build of VyOS, you will need to build it yourself.

This will work just fine on any modern Linux distribution–even ones running inside of WSL2 on Windows 11.

Build steps

  1. Install podman
sudo apt install podman
  1. Pull the build environment w/ podman
sudo podman pull docker.io/vyos/vyos-build:equuleus
  1. Fetch source code from github
git clone -b equuleus --single-branch https://github.com/vyos/vyos-build
  1. Enter build environment
cd vyos-build
sudo podman run --rm -it --privileged -v $(pwd):/vyos -w /vyos vyos/vyos-build:equuleus bash
  1. Build it
./configure --architecture amd64 --build-by "hello@example.com"
sudo make iso

Once finished, exit the container. Inside the “build” directory will be the .iso image you’ll need to boot in your hypervisor.
